## 📋 О проекте
Приложение созданное с помощью Java Script, CSS, HTML для шифрования и дешифрования файлого любого формата
на основе **алгоритма Эль-Гамаля**.

---
## 🎮 Инструкция по использованию

### Шаг 1. Введите простое число *p*.
---
### Шаг 2. Выберите первообразный корень *g* по модулю числа *P* 
---
### Шаг 3. Введите закрытый ключ *x*.
Далее вычисляется открытый ключ *y* по формуле:
***y* = *g*^*x* mod *p***
---
### Шаг 4. Введите случайное число *k*, такое что принадлежит промежутку (1, *p*-1) и взаимнопростое с *p-1*.
---
### Шаг 5. Выберите файл
---
### Шаг 6. Нажмите конпку "Зашифровать"/"Расшифровать".
